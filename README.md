# Project Description
The project aims at making an easy approach towards tracking events taking place within an institution such as a learning institution easily.

Events page where the events are easily viewed by the institution
The user is able to add events
Countdown feature to tell how many days and hours remain for the day of the event thus enhancing easy planing.
